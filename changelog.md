# Changelog

All notable changes to this project will be documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 16-04-2026

### Added

- List
- ImmutableList
- BaseSet
- ImmutableSet
- BaseMap
- ImmutableMap

## [0.0.2] - 16-04-2026

### Fixed

- fixed typos in readme

## [0.0.3] - 16-04-2026

### Added

- `ofIterable(iterable)` to create a list/set from an iterable
- `ofLists(lists)` to create a list from lists
- `ofSets(sets)` to create a set from sets

### Fixed

- fixed more typos in readme
- added changelog

## [0.0.4] - 16-04-2026

### Added
- reverse() to reverse the order of files in the list/set
- copy() to create a new copy of the list/set

## [0.0.5] - 01-05-2026

### Added
- sort(predicate) to sort the list/set(works exactly like the array implementation);
- shuffle() to shuffle the items in the list/set/map
- getRandomItem() to get a random item from the list/set

### Fixed
- fixed typos in changelog
